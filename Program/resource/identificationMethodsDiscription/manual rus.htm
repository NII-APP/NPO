Метод выделения пика

Данный метод позволяет идентифицировать модальные параметры для одного резонансного пика.

Последовательность действий

  1. Необходимо выбрать диапозон частот, в котором будет происходить поиск резонансной частоты. В процессе выбора значения найденной резонансной частоты автоматически обновляется.
  2. Пользователь имеет возможность вручную скорректировать значение найденной резонансной частоты.
